## 下载安装依赖

```shell
git clone https://github.com/jackchen0120/web3-dapp-all-example.git
cd vue3-dapp-todo
yarn or npm install
```

## 开发模式

```shell
yarn dev
```

运行之后，访问地址：http://localhost:5173/

## 生产模式

```shell
yarn build
```

## 测试模式
